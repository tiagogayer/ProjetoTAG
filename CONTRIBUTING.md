# Contributing

Bug reports, technical suggestions, and general feedback are always welcome.
[Open an issue](https://github.com/tiagogayer/ProjetoTAG/issues/new/choose)
to get started — templates are available to guide your report.

Thank you for taking the time to reach out. Pull requests are not the expected
external channel: if you spotted something, an issue is all that is needed.
The maintainer will review it and apply any fix directly.
Every report is genuinely appreciated.
